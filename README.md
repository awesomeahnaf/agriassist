# AgriAssist - কৃষি সহায়তা

AgriAssist is a web-based agricultural advisory and crop problem case tracking system for farmers in Bangladesh.

## How to Run Locally

This project is built with **React**. It uses your browser's **localStorage** to save data, so you don't need a backend server or database setup.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (comes with Node.js)

### Installation & Running

1. Download and extract the code.
2. Open your terminal or command prompt in the project folder.
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm run dev
   ```
5. Open your browser and go to: `http://localhost:5173` (or the URL shown in your terminal).

## Features

- **Farmer Dashboard:** Submit crop problems and track status.
- **Field Officer:** Provide video/audio advice to farmers.
- **Expert Dashboard:** Diagnose problems and provide solutions.
- **Bangla Language Support:** Full interface in Bangla (বাংলা).
- **Offline Storage:** Data is saved in your browser automatically.
